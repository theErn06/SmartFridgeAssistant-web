A Capstone Project (TARUMT KL) by 4 Year 3 students 

Abstract

Household food waste remains a critical global issue, caused by inefficient consumer inventory management. Users do experience forgetfulness and difficulty locating items within their refrigerators, resulting in spoilage and unnecessary repurchases. While smart appliances exist with embedded artificial intelligence, cloud dependency introduces privacy risks, latency, and subscription costs. To address these challenges, this capstone project proposes an offline, AI-powered smart refrigerator assistant capable of providing hands-free inventory tracking and intelligent item management to effectively minimise household food waste. To achieve the primary objective of this project, the system’s methodology is centred around a custom Python-based execution engine that controls the entire offline processing pipeline. An open-source Edge-Large Language Model (Edge-LLM) from Ollama, acting as a "Jarvis" reasoning engine, translates voice commands via Voice Activity Detection (VAD) and a Speech-to-Text (STT) model into deterministic JSON to execute local database operations. A dynamic webpage synchronises with this database for live tracking, expired statuses, and waste reports across the local network. The final hardware design integrates an edge host with an Arduino, a door sensor, and an LCD for immediate multimodal feedback. Evaluation showed a 92.92% response accuracy with a response time (speed) of around 3 seconds demonstrating the system’s effectiveness for conventional refrigerators in smart tracking and waste reduction.
